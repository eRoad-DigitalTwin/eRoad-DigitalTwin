eRoadMontBlanc

Digital Twin Platform

ABOUT

Supervision

Our digital twin platform enables real-time monitoring and analysis through dashboards, maps and 2D/3D visualizations.

Interaction

Data analysis allows automated or assisted actions on the physical system when predefined conditions or thresholds are reached.

Simulation

Extracted data can be used to run simulations and study system behavior under various parameters and scenarios.

DEMONSTRATION

I. DASHBOARDS

Create custom dashboards by adding available components, resizing them and arranging them freely.

Access model-related charts and indicators directly from the navigation bar.

II. 2D / 3D VISUALIZER

Visualize the physical environment using an interactive 2D/3D engine. Filter elements, switch display modes and interact with objects to access contextual information.

III. DATA EXTRACTION

Easily extract geographic data by selecting routes or points of interest directly on the map.

Generated datasets can be configured, exported and reused within models and simulations.

IV. MODELS

Manage and use multiple domain-specific models directly within the platform.

Models can be added, configured and selected to address different simulation objectives.

V. SIMULATOR

Launch simulations by combining extracted data, models and user-defined parameters.

Simulations can be executed globally or on a segment-by-segment basis.

VI. SCENARIOS

Create, save and replay simulation scenarios to compare different configurations.

Results are stored and accessible for analysis and decision support.

VII. DATA SOURCES

Add and manage external data sources to enrich models and simulations.

Data sources can be tested and reused without impacting the rest of the platform.
